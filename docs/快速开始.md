# 快速开始指南

## 第一步：环境准备

```bash
# 1. 创建虚拟环境
python -m venv venv

# 2. 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key
```

## 第二步：理解项目结构

打开以下文件，理解项目架构：

1. **`docs/架构设计.md`** - 整体架构设计
2. **`docs/学习路径.md`** - 分阶段学习计划
3. **`TODO.md`** - 详细的开发任务列表

## 第三步：开始实现

### 阶段一：基础 RAG（推荐从这里开始）

1. **实现 LLM 模块**
   - 打开 `src/core/llm/base.py`
   - 阅读 TODO 注释
   - 实现 `OpenAILLM` 类
   - 参考：[LangChain OpenAI 文档](https://python.langchain.com/docs/integrations/chat/openai)

2. **实现 Embedding 模块**
   - 打开 `src/core/embedding/base.py`
   - 实现 `OpenAIEmbedding` 类
   - 参考：[LangChain Embeddings 文档](https://python.langchain.com/docs/integrations/text_embedding/)

3. **实现向量存储模块**
   - 打开 `src/core/vectorstore/base.py`
   - 实现 `FAISSVectorStore` 类
   - 参考：[LangChain FAISS 文档](https://python.langchain.com/docs/integrations/vectorstores/faiss)

4. **实现 RAG Chain**
   - 打开 `src/core/chain/rag_chain.py`
   - 这是核心部分，仔细实现
   - 参考：[LangChain RAG 教程](https://python.langchain.com/docs/use_cases/question_answering/)

5. **测试**
   - 使用 `examples/basic_rag_example.py` 测试你的实现

## 开发建议

### ✅ 应该做的

1. **先理解再实现**
   - 阅读 LangChain 官方文档
   - 理解每个组件的职责
   - 查看示例代码

2. **逐步测试**
   - 每实现一个模块就测试
   - 使用简单的测试用例
   - 打印中间结果验证

3. **记录问题**
   - 遇到问题记录下来
   - 查找解决方案
   - 总结经验和教训

4. **保持代码整洁**
   - 添加注释
   - 遵循代码规范
   - 及时重构

### ❌ 不应该做的

1. **不要跳过基础**
   - 不要直接复制粘贴代码
   - 不要跳过理解环节
   - 不要急于完成

2. **不要一次性实现所有功能**
   - 按照 TODO 列表逐步实现
   - 每个阶段完成后再进入下一阶段

3. **不要忽视错误处理**
   - 添加适当的异常处理
   - 提供有意义的错误信息

## 遇到问题？

1. **查看文档**
   - LangChain 官方文档
   - LangGraph 官方文档
   - 项目内的 `docs/` 目录

2. **调试技巧**
   - 打印中间结果
   - 使用断点调试
   - 逐步测试每个组件

3. **寻求帮助**
   - GitHub Issues
   - Stack Overflow
   - LangChain 社区
